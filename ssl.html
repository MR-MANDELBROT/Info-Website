Beim Online-Shopping kann man Heutzutage seine Kreditkarteninformation relativ sicher verwenden, aber warum ist das überhaupt so?

Das liegt am Verschlüsselungsverfahren SSL oder Secure Socket Layer.


Bei vielen Browsern wird in der Suchleiste ein kleines Schloss angezeigt. So sieht das bei mir aus!

Wenn ich es Anklicke Erscheint ein Banner.

Hier ist von einem Zertifikat die Rede.



Aber was ist so'n Zertifikat?

Identifikation

Public Key

Signature

Ein Zertifikat Setzt sich aus drei Elementen zusammen.

Die Identifikation enthält Informationen über den Besitzer des Zertifikats, wie zb. den Domainnamen.


Aber was ist so'n Public Key?

Ein Public Key ist teil eines Schlüsselpaars und hat einen korrespondierenden Private Key.


Public Key

Jeder hat Zugriff auf den Public Key.

Private Key

Nur der User hat Zugriff auf den Private Key.

Bei dem Schlüsselpaar handelt es sich um eine Assymetrisch Verschlüssellung.

Das bedeutet alles was der Private Key verschlüsselt kann nur vom Public Key entschlüsselt werden und umgekehrt.

Eine Nachricht die Alice mit dem Public Key Verschlüsselt kann dementsprechent nur von Bob entschlüsselt werden, der den Private Key besitzt.


Von wem wird das Zertifikat unterschrieben?

Eine Certificate Authority ist für das Unterschreiben der Zertifikate zuständig.


Certificate Authority (CA)
  • Hat ein Schlüsselpaar

  • Den Private Key hat nur die CA. selbst

  • Jeder Browser hat eine Liste mit den Public Key's der CA's denen er vertraut

Die Signiture ist mit dem Private Key der CA. verschlüsselt.

Der Browser kann mit dem Public Key aus seiner Liste der Vertrauenswürdigen CA's also die Signature zweifelsfrei entschlüsseln.

Dein Browser weiß jetzt wegen der Signature der CA. also das www.youtube.com wirklich YouTube ist und hat den Public Key von YouTube.

Dein Browser kann mit dem Public Key jetzt alles verschlüsseln, so dass es nur von www.youtube.com mit dem Private Key entschlüsselt werden kann.

Dein Browser generiert jetzt einfach einen neuen Schlüssel der sowohl verschlüsseln als auch entschlüsseln kann (Symmetrische Verschlüssellung) und verschlüsselt ihn mit dem Public Key.

Dein Browser

Neuer Geheimschlüssel!

www.youtube.com

Der Client schickt den neuen Schlüssel an den Server.

Der Client und Server sind die einzigen zwei Geräte im ganzen Internet mit dem neuen Schlüssel und können damit ihren gesamten Datenverkehr verschlüsseln.
